# Mohamed Abdelaziz | DsMans0021

A modern, responsive personal portfolio website built with pure HTML, CSS, and JavaScript.

## 🌐 Live Site
- **Main Portfolio**: [https://dsmans0021.github.io](https://dsmans0021.github.io)
- **Tools Page**: [https://dsmans0021.github.io/tools.html](https://dsmans0021.github.io/tools.html)

## ✨ Features

### Design & UX
- **Dark/Light Theme Toggle**: Seamless theme switching with smooth transitions
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Clean, minimalist design with custom CSS variables
- **Smooth Animations**: Hover effects, transitions, and micro-interactions

### Technical Stack
- **Pure HTML5**: Semantic markup and accessibility features
- **Custom CSS3**: Advanced styling with CSS Grid and Flexbox
- **Vanilla JavaScript**: No framework dependencies for optimal performance
- **Google Fonts**: Inter and Space Mono for typography
- **Font Awesome Icons**: Comprehensive icon library

### Pages
- **Home Page** (`index.html`): Main portfolio with projects, about section, and contact information
- **Tools Page** (`tools.html`): Collection of useful tools and utilities

## 🎨 Theme System
The website features a sophisticated theming system with:
- CSS custom properties for consistent color schemes
- Dark mode with green accent colors (`#39ff14`)
- Light mode with complementary colors
- Smooth theme transitions

## 🚀 Performance
- **No Build Process**: Direct HTML/CSS/JS for fast loading
- **Minimal Dependencies**: Only external fonts and icon libraries
- **Optimized Assets**: Efficient CSS and JavaScript
- **SEO Friendly**: Proper meta tags and semantic HTML

## 📱 Browser Support
- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile-responsive design
- Touch-friendly interactions

## 🔧 Customization
The website uses CSS custom properties for easy theming and customization. Key variables include:
- `--bg`, `--bg2`: Background colors
- `--surface`, `--surface2`: Card and panel backgrounds
- `--text`, `--text-strong`: Typography colors
- `--accent`, `--accent2`: Highlight and interaction colors
